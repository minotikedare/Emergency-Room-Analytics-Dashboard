# Emergency-Room-Analytics-Dashboard

This project presents an interactive Emergency Room (ER) Analytics Dashboard designed to monitor patient flow, track wait times, analyze demographics, and generate actionable operational insights for hospital management.

---

## Project Overview
Emergency Rooms handle thousands of patients with varying urgency levels. Efficient monitoring of wait times, referrals, and admission patterns is crucial for improving patient experience and resource planning.

This dashboard provides:
- Real-time visibility into ER performance  
- Data-driven insights for decision-making  
- Comprehensive patient analytics  

---

## Objectives
- Analyze ER patient trends and service utilization  
- Monitor average wait times and identify operational bottlenecks  
- Track demographic distribution of patients  
- Evaluate referral patterns across departments  

---

## Resetting Dashboard View

If filters are applied and you want to return to the original default dashboard view:

**Mac Users**
- Click the **Revert** icon from the top toolbar or Press **Command + Z** multiple times

**Windows Users**
- Click the **Revert** icon from the top toolbar or Press **Ctrl + Z** multiple times

If a Reset button is provided below the dashboard, simply click it to restore the original view.
